Mobile Friendly Twenty Ten helps make websites and blogs running the Twenty Ten Theme - responsive.

== Description ==

Mobile Friendly Twenty Ten helps make websites and blogs running the Twenty Ten Theme - responsive.
This is done by installing and moving from Twenty Ten theme to Twenty Eleven, which by installing the
Mobile Friendly Twenty Ten plugin is made to much look like Twenty Ten.
Since Twenty Ten is fully responsive, the resulting website is sort of a responsive Twenty Ten theme.

== Installation ==

1. Upload and select the Twenty Eleven theme through the 'Design' menu and 'Themes' submenu in WordPress.
2. Upload and activate the plugin through the 'Plugins' menu in WordPress
